# Data Sources & Methodology

This directory contains the datasets used by the Uzbekistan Payment Volume Forecasting project.

---

## Files

### `uzbekistan_payments.csv`
Synthetically generated daily payment transaction data for Uzbekistan, 2019–2024 (2,192 rows).

Generated by `data/generate_data.py`. Calibrated to aggregate figures published in CBU Statistical Bulletins.

| Column | Description |
|---|---|
| `date` | Calendar date (daily) |
| `transaction_volume` | Estimated daily digital payment transactions |
| `avg_transaction_value` | Average transaction value in UZS (thousands) |
| `region` | Simulated region (Tashkent, Samarkand, Fergana, Namangan, Andijan) |
| `day_of_week`, `month`, `year`, `day_of_month`, `quarter` | Calendar features |
| `is_ramadan`, `is_navruz`, `is_payday` | Uzbekistan-specific event flags |

Key patterns encoded in the synthetic data:
- **Exponential growth** — 500K → 3.6M daily transactions (2019–2024)
- **Structural breaks** — FPS launch 2021 (×1.35), Uzum Bank launch 2023 (×1.20)
- **Weekly seasonality** — weekends ~22% lower
- **Ramadan** — ~12% reduction (dates shift each year per Islamic calendar)
- **Navruz** — March 20–23, ~35% increase (national holiday)
- **Payday** — 10th and 25th of month, ~22% spike
- **COVID-19** — Q2 2020, ~35% reduction

---

### `exchange_rates_cbu.csv`
Real USD/UZS exchange rates fetched from the Central Bank of Uzbekistan's public API, 2019–2024.

Fetched by `data/fetch_exchange_rates.py`.

| Column | Description |
|---|---|
| `date` | Calendar date |
| `usd_uzs_rate` | Official USD/UZS exchange rate (CBU) |

**Source:** `https://cbu.uz/en/arkhiv-kursov-valyut/json/{date}/`

Exchange rate range: ~8,400 → ~12,700 UZS per USD (2019–2024).

---

### `uzbekistan_payments_enriched.csv`
The main dataset used for training and evaluation. Merges `uzbekistan_payments.csv` with `exchange_rates_cbu.csv`, adding:

| Column | Description |
|---|---|
| `usd_uzs_rate` | Daily CBU exchange rate |
| `usd_uzs_rate_lag7` | Exchange rate 7 days prior |
| `usd_uzs_rate_change` | 7-day percentage change in exchange rate |

This is the file consumed by `src/train.py` and `src/predict.py`.

## Regenerating the Data

The CSV files are already included in the repository. To regenerate from scratch:

```bash
# Step 1: Generate synthetic payment volumes
python data/generate_data.py

# Step 2: Fetch real exchange rates from CBU API (~3 minutes, parallelized)
python data/fetch_exchange_rates.py
```

> **Note:** `fetch_exchange_rates.py` makes ~2,192 API calls to `cbu.uz`. It uses
> 20 parallel threads and takes approximately 2–4 minutes on a normal connection.

---

## Data Disclaimer

Transaction volume data is **synthetically generated** and calibrated to publicly
available CBU aggregate statistics. It does not represent actual transaction records
from any payment platform. USD/UZS exchange rates are sourced directly from the
Central Bank of Uzbekistan's public API and are genuine historical data.
